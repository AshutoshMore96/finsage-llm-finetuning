Architecture

Fine-Tuned Foundation

QLoRA (4-bit NF4) adapters trained on ~20K financial instruction pairs, then DPO preference alignment using a self-generated preference dataset — the RLHF "data flywheel".

3B params · 4-bit QLoRA + DPO

Domain Expertise

Trained on SEC 10-K Q&A, broad finance instructions, and FinQA numeric reasoning (Sujet-Finance-Instruct, financial-qa-10K, FinQA). Evaluated on a held-out set with a debiased LLM judge.

ROUGE-L 0.33 → 0.66

Production Ready

Merged and quantised to GGUF (Q4_K_M, 1.8 GB). Runs locally on an 8 GB Apple M2 via Ollama / llama.cpp, or as a FastAPI service — Docker-ready for any cloud.
